####################################

This folder and its are only used if the app view is launched

####################################

If you want to set your own icon for a FILE TYPE, name it using this pattern:

(file extension).png

Example:
1. md.png
2. txt.png
3. docx.png

####################################

If you want to bind your own to a FILE NAME, name it using this pattern:

(file name).(file extension).png

Example:
1. info.md.png
2. startup.docx.png
3. chrome.exe.png

####################################

If you want to set your own icon for a FOLDER, name it using this pattern:

(folder name).dir.png

Example:
1. img.dir.png
2. src.dir.png
3. System32.dir.png

####################################

Have a nice time using the app!
